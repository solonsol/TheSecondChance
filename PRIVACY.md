# Operator vs. visitor (this repo)

**Nothing in the public HTML identifies who built or runs the site** — no personal name, email, wallet, or `mailto:`.

The **Instagram embed** shows a **third-party post** (whoever owns that IG account). That does not reveal the site operator unless you choose to link those identities elsewhere (same bio, same wallet, etc.).

**What still can tie you to the site (outside these files):**

- Git commit author email / name if you push to a public repo  
- Domain registrar, hosting, and payment records  
- Reusing wallets, socials, or IPs across “personal” and “project”  

This repo only controls the **static page content** — keep deploy and Git accounts separate from your real identity if that’s your goal.
